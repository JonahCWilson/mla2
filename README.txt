1. Install jupyter notebooks : https://jupyter.org/install
2. Install scikit-learn: https://scikit-learn.org/stable/install.html
3. Install numpy and matplotlib: https://www.scipy.org/install.html
4. Install mlrose (hiive fork) : pip3 install mlrose-hiive
5. Install mlrose standard : pip3 install mlrose
6. Clone https://github.com/JonahCWilson/mla2.git into a directory of your choosing.
5. run "jupyter notebook Assignment2.ipynb"
6. A browser window should pop up.  Once it does, run all cells
